what it does

new

surprise-based importance

every new memory is compared against existing embeddings using k-NN. novel memories get importance boosted, redundant ones get flagged. inspired by the Titans paper.

new

retention regularization

three forgetting modes: L2 (classic exponential), Huber (robust to burst-then-quiet), elastic net (sparse retention). access reinforcement adds spaced repetition.

new

deep retrieval reranker

2-layer MLP trained on actual access patterns. learns which memories get used after being returned. persists to disk, runs automatically.

new

cross-domain synthesis

dream cycle finds entity pairs in different domains with moderate embedding similarity. LLM confirms genuine connections. creates bridge memories.

new

cognitive scaffolding

recall_hints returns truncated snippets + entity names to trigger recognition without replacing cognition. the agent decides when to pull full context.

core

5 memory layers

working (ephemeral) → episodic (experiences) → semantic (permanent) → procedural (decisions, errors) → codebase (compressed code). memories promote upward when they prove useful.

core

entity graph

auto-extracts people, tools, projects from every memory. relationship graph with co-occurrence strength. multi-hop traversal via recursive SQL CTEs.

core

dream cycle

consolidation pass: cluster similar memories, summarize, generate entity peer cards, archive low-value old stuff. like sleep for your memory system.

core

neural visualization

force-directed entity graph with concentric layer rings. neurons fire with traveling impulse particles when memories are accessed. polls the database across processes.

built on research

Titans

Neural Long-Term Memory (Behrouz et al., 2025)

surprise-based memorization where memory updates are proportional to the gradient of loss. engram's k-NN novelty scoring at write time comes from this.

Miras

Unifying Framework for Sequence Models (Behrouz et al., 2025)

shows forgetting mechanisms are retention regularization. engram's three decay modes (L2, Huber, elastic net) come from this framework.

EEG study

Your Brain on ChatGPT (Kosmyna et al., 2025)

LLM users develop weaker neural connectivity when given full context. engram's recall_hints returns partial context to trigger recognition, not replace it.
